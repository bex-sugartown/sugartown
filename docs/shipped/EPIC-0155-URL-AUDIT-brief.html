Overview
What
Review the 266 legacy WordPress taxonomy-namespace URLs surfaced in the URL audit artifacts. Make keep / redirect / drop decisions for each group. Then wire those decisions into the existing redirect infrastructure so the _redirects file and Sanity redirect documents are authoritative and complete before cutover.
Why
Redirect infrastructure already exists — redirect schema, redirects.js migration script, build-redirects.js, and apps/web/public/_redirects are all live. What's missing is editorial decisions on the 266 URLs. Without those, the launch routing epic cannot be scoped or started. Every day these are undecided is a day of SEO equity at risk on cutover.
Who Does What
Bex — Phase 1: reviews audit artifacts, makes routing decisions, fills in the decision registry CSV. No code required. Claude Code — Phase 2: implements approved decisions into Sanity redirect documents and _redirects file, runs validation.
Definition of Done
Every one of the 266 URLs has an explicit decision (redirect / drop / keep). All redirect decisions are implemented in the Sanity redirect schema and _redirects file. pnpm validate:urls passes clean. Launch routing epic can begin.

Decision Framework — How to Evaluate Each URL
301 REDIRECT — Preserve
URL has a meaningful canonical equivalent in the new system. Preserves SEO equity. Use for: category/tag archives → new taxonomy URLs, post ID URLs → article slugs, any URL with inbound links or search indexing worth keeping.
410 GONE — Drop
URL has no equivalent and no SEO value worth preserving. 410 (Gone) is better than 404 for intentional deletions — tells crawlers it's gone for good. Use for: /wp-admin, /wp-login, RSS feeds without replacements, WP cruft paths.
KEEP — No Change
URL path is identical in the new system and content is already mapped. No redirect rule needed. Confirm the slug is clean and live in Sanity. Use for: pages that kept their exact slug through migration.
DEFER — Needs More Info
Decision requires data you don't have yet (e.g. which WP media URLs are linked from outside, whether /author pages are indexed). Flag it, note what info is needed, and return after gathering. Don't let defers block Phase 2 from starting on clear decisions.

  • Every redirect has a destination URL and a status code (301 or 302)

Done When
  • Sanity redirect documents created for all 266 decisions (or confirmed already present from migration script)
  • _redirects file regenerated from complete Sanity document set
  • No conflicts between new decisions and existing redirect documents
  • No duplicate source paths in _redirects
  • pnpm validate:urls passes clean
